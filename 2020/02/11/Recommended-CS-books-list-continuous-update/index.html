Recommended CS books list[under continuous update]

计算机科学推荐书单[持续更新…]


编码

  • CODE: The hidden Language of Computer Hardware and Software
  • Charles Petzold
  • 4.5 star
  • 计算机体系结构入门神书，从莫尔斯码讲到盲文，用继电器搭起一台计算机

代码整洁之道 1

  • Clean Code: A Handbook of Agile Software Craftsmanship
  • Robert C.Martin
  • 3.8 star
  • 很好的一本编码习惯经验总结，代码如何命名，如何写好注释，如何设计函数、类，整个系统应该怎样迭代演进，值得一读

代码整洁之道 2

  • The Clean Coder: A Code of Conduct for Professional Programmers
  • Robert C.Martin
  • 3.5 star
  • 一本简短的册子，虽然与技术无关，但是很好地讲述了程序员应该如何让自己更加专业，怎样专业地面对需求，如何正确处理项目的延期和交付，如何验收和测试，怎样管理时间，管理自己的压力，如何与别人协作

设计模式: 可复用面向对象软件的基础

  • Design Pattern: Element of Reusable Object-Oriented Software
  • Erich Gamma/ Richard Helm/ Ralph Johnson/ John Vlissides
  • 4.6 star
  • 设计模式只看这本，反复看就完事了

LINUX系统编程

  • LINUX SYSTEM PROGRAMMING
  • Robert Love
  • 3.6 star
  • 这是一本极好的linux syscall api reference book，适合当工具书

深入浅出MFC

  • Dissecting MFC —using Visual C++ 5.0 & MFC4.2
  • 候杰
  • 4.2 star
  • 学MFC时的第一本书。十多年过去了，候杰老师的书透彻清晰，印象深刻

Java编程思想

  • Thinking in Java
  • Bruce Eckel
  • 4.6 star
  • 学Java看这本就完事了

C++ Primer Plus

  • C++ Primer Plus
  • Stephen Prata
  • 3.8 star
  • 浅显通俗易懂，C++入门必备

编程珠玑

  • Programming Pearls
  • Jon Bentley
  • 4.5 star
  • 编程像搞艺术？请看这本

算法导论

  • Introduction to Algorithms
  • Thoms H.Cormen / Charles E.Leiserson / Ronald L.Rivest / Clifford Stein
  • 4.8 star
  • 第一次看是2011年8月27日，18岁的生日。9年后学到的道理就是人生得耐心，推导要慢慢啃

Linux程序设计

  • Beginning Linux Programming
  • Neil Matthew / Richard Stones
  • 3.5 star
  • 一本浅显易懂的Linux系统编程入门书，丐版APUE

深入理解计算机系统

  • Computer System: A Programmer’s Perspective
  • Randal E.Bryant / David O’Hallaron
  • 4.3 star
  • 程序员视角讲体系结构，程序优化必读

推荐系统实践

  • Recommand System
  • 项亮
  • 3.2 star
  • 还行吧

深度学习的数学

  • Deep learning ga wakaru sugaku nyumon
  • 涌井良幸 涌井贞美
  • 3.9 star
  • 虽然内容有点过于简单，但是整数简洁明了、循序渐进，讲了深度学习的初等预备知识，看得很爽

Python在机器学习中的应用

  • Python在机器学习中的应用
  • 余国本 孙玉林
  • 2.9 star
  • 本身是本拼凑式blog，之所以还给分是因为章节铺的很好，可以了解机器学习的知识体系

Python神经网络编程

  • Make Your Own Neural Network
  • Tariq Rashid
  • 2.7 star
  • 不如深度学习的数学

ROS机器人开发实践

  • ROS机器人开发实践
  • 胡春旭
  • 2.9 star
  • 关于ros开发一本不错的blog集

Software Architecture for Developers

  • 软件架构
  • Simon Brown
  • 3.0 star
  • 感觉就是一堆培训讲义
